# 项目结构

GitHub Pages 由根目录的 `index.html` 提供，入口必须保留在这里。

- `assets/jinzhu/`：金主的透明多帧网页动画素材。
- `assets/jinzhu-home/`：网页内的家居小道具。
- `styles/`：主页面样式。
- `scripts/`：时钟、天气、留言板、金主作息和本机储存逻辑。
- `legacy/message/`：旧的独立留言管理页，已和主页面分开，保留作兼容用途。
- `desktop-overlay/`：独立桌面覆盖层项目，不属于 GitHub Pages 主页面。
- `KY-Workspace/`、`KY-Asset-Lab/`、`outputs/`：本机工作资料/产物，不属于网页发布内容，未作移动或删除。
